### Executando o Projeto
Siga os passos abaixo para executar os testes e gerar relatórios:

#### Instale as Dependências:

```bash
npm install
```

#### Execute os Testes e Gere os Relatórios:

```bash
npm run test:report
```

Acesse o relatório em: `cypress/reports/html/index.html`
